_exception_handler¶

Exception Handler

This is the custom exception handler that is declaired at the top of Codeigniter.php. The main reason we use this is to permit PHP errors to be logged in our own log files since the user may not have access to server logs. Since this function effectively intercepts PHP errors, however, we also need to display errors based on the current error_reporting level. We do that with the use of a PHP error template.
